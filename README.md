# Gericht - Landing Page

Landing page for Gericht, a fine dining restaurant which offers a memorable dining experience.

## Technology stack

- React 18
- CSS3 (BEM Model)

## Features

- Mobile responsive
- Uses modern design principles
- Smooth transitions and gradients
